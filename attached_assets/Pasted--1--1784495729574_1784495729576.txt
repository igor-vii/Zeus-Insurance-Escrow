Тема: Заменить заглушки на реальную логику в MCP-сервере для эскроу

Шаг 1: Обновить api-server/src/services/escrow.ts
Замени заглушки на реальные вызовы SDK. Используй существующий ZeusSDK и методы для эскроу на Base Mainnet.

Пример для prepareCreateEscrowCalldata:

typescript
import { ZeusSDK } from "@zeus/sdk";
import { ethers } from "ethers";

export async function prepareCreateEscrowCalldata(
  executor: string,
  amount: string,
  timeout: number
) {
  // 1. Инициализируем SDK с серверным кошельком
  const provider = new ethers.JsonRpcProvider(
    process.env.RPC_URL_MAINNET || "https://mainnet.base.org"
  );
  const signer = new ethers.Wallet(process.env.SERVER_PRIVATE_KEY!, provider);
  const sdk = new ZeusSDK();
  await sdk.connect("base-mainnet", signer);

  // 2. Генерируем calldata для создания сделки
  const { agreementId, fee, calldata } = await sdk.escrow.prepareCreateAgreement(
    executor,
    BigInt(amount),
    timeout
  );

  return {
    agreementId: agreementId.toString(),
    fee: fee.toString(),
    calldata: {
      to: calldata.to,
      data: calldata.data,
      value: calldata.value || "0",
    },
  };
}
Аналогично обнови:

prepareConfirmEscrowCalldata → sdk.escrow.prepareConfirm(agreementId, proof)

prepareRefundEscrowCalldata → sdk.escrow.prepareRefund(agreementId)

getEscrowAgreementById → запрос к БД policies_cache (используй Drizzle)

json
{
  "dependencies": {
    "@zeus/sdk": "file:../sdk"
  }
}
