Шаг 2: Закоммитить изменения
bash
git add agent.json

Шаг 3: Зарегистрировать протокол в реестре ERC-8004
Вызови функцию register(string agentURI) контракта 0x8004A169FB4a3325136EB29fA0ceB6D2e539a432 на Base Mainnet:

bash
cast send 0x8004A169FB4a3325136EB29fA0ceB6D2e539a432 \
  "register(string)" \
  "https://raw.githubusercontent.com/igor-vii/Zeus-Insurance-Escrow/main/agent.json" \
  --private-key $PRIVATE_KEY \
  --rpc-url https://mainnet.base.org
Или через Hardhat:

bash
cd contracts
npx hardhat run scripts/register-erc8004.ts --network base-mainnet
